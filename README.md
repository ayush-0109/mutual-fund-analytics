# Mutual Fund Analytics

## Project Overview

Mutual Fund Analytics is a data analytics project that analyzes mutual fund performance, investor transactions, NAV history, SIP inflows, and risk metrics using Python, SQLite, and SQL.

## Objectives

- Perform ETL on mutual fund datasets
- Validate and clean raw data
- Load datasets into SQLite database
- Design star schema for analytics
- Perform exploratory data analysis (EDA)
- Analyze AUM, SIP inflows, and risk metrics
- Generate insights using SQL queries
